# Grading System (Java)

A Java-based application to calculate and assign grades to students.

## Features
- Stores student marks using arrays
- Batch processing of student data
- Grade calculation using conditional logic
- Modular code for input handling and result display

## Technologies Used
- Java
- Arrays
- Conditional Statements
- Functions / Methods

## How to Run
javac GradingSystem.java
java GradingSystem
